# c_basce_addressbook
Poject Name :- Addressbook

Description :- This is a console-based Address Book application implemented in C
for contact managemetn or arragement.
It uses file handling and string, pointers, structer data type, loops and functions.

Features :- 
    - create new contact
    - List all contacts
    - Search contact by name or number
    - Edit existing contact
    - Delete contact
    - Save contacts to file
    - Load contacts from file

 Technologies Used:-
      - C Programming Language
      - File Handling
      - string
      - user define funtions
      - struncture
